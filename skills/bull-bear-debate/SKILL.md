# 多视角辩论框架

## 第一轮：Bull Case
以乐观投资者的视角分析目标公司。重点关注：
- 增长潜力和市场机会
- 市场地位和竞争优势
- 技术壁垒和创新能力
- 财务健康度和盈利趋势

每个论点附具体数据来源。

## 第二轮：Bear Case
以怀疑论者视角分析目标公司。重点关注：
- 估值泡沫风险
- 竞争威胁和市场份额侵蚀
- 周期性下行和宏观逆风
- 监管风险和地缘政治风险
- 管理层执行力和治理问题

每个论点附具体数据来源。

## 第三轮：裁判
综合以上两个视角，给出权衡后的结论：
- 哪些 bull/bear 论点更有说服力，为什么
- 整体置信度评级：强看多 / 弱看多 / 中性 / 弱看空 / 强看空
- 关键转折点：什么事件/数据会让你改变判断
- 需要人工验证的关键假设清单

## 输出
将辩论记录写入 `research/decisions/{TICKER}-bull-bear-{DATE}.md`
